# 紳士打碼 v1.0.0

![紳士打碼 Logo](./logo-Photoroom.png)

## 介面演示

![UI Demo](./UI.PNG)

單檔可直接開啟的圖片打碼工具（`standalone.html`），支援馬賽克、海苔條、不規則塗抹、框選、歷史回溯、預覽、縮放檢視，以及中英文與深色模式切換。

## 快速開始

### 直接使用（免安裝）
1. 進入專案根目錄。
2. 直接用瀏覽器開啟 `standalone.html`。
3. 載入圖片後即可編輯與下載。

### React/Vite 開發模式
1. 安裝依賴：
   ```bash
   npm install
   ```
2. 啟動開發伺服器：
   ```bash
   npm run dev
   ```
3. 打包：
   ```bash
   npm run build
   ```
4. 預覽打包結果：
   ```bash
   npm run preview
   ```

## 主要功能

- 打碼效果
  - 馬賽克
  - 海苔（可調顏色、透明度、寬度、間隔、方向）
- 選取模式
  - 塗抹（不規則軌跡，支援圓形/方形筆刷）
  - 框選（拖拉矩形）
- 影像檢視
  - 滾輪縮放
  - 右上重置縮放並置中
  - 右上顯示目前縮放百分比
- 工作流
  - 待處理區域先預覽，再按「確定套用」
  - 上一動作 / 下一動作
  - 歷史清單回溯
  - 下載 PNG
- 介面
  - 中文 / English 切換
  - Light / Dark 切換

## 參數說明

- 顆粒尺寸：`1 ~ 30 px`
  - 註：為了確保可見效果，UI 顯示 `1px` 時會套用「最小可見馬賽克」。
- 筆刷大小：`4 ~ 500 px`
- 海苔透明度：`0 ~ 100%`
- 海苔寬度：`2 ~ 200 px`
- 海苔間隔：`0 ~ 120 px`
- 海苔方向：水平 / 垂直

## 使用流程（建議）

1. 上傳圖片。
2. 選擇打碼效果（馬賽克或海苔）。
3. 選擇模式（塗抹或框選）。
4. 調整參數（顆粒、筆刷、海苔等）。
5. 在圖片上選取要處理區域。
6. 先看右側效果預覽，確認後按「確定套用」。
7. 需要時使用 Undo/Redo 或歷史回溯。
8. 按下載輸出最終圖片。
